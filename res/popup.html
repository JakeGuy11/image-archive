Overview

Image Sourcerer is a browser extension that allows the user to download, organize and track images from supported sites.

Instructions

Whenever you see a , you can click it to download the image. You will be prompted for a name and/or a path. Specify the path using forward slashes (starting without a slash). Do not include the file extension.
